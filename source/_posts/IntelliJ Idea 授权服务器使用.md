### JetBrains授权服务器

``` groovy
http://intellij.mandroid.cn/

备用地址
http://idea.wrbug.com/
```


### 支持的版本
IntelliJ IDEA 7.0 或更高
ReSharper 3.1 或更高
ReSharper Cpp 1.0 或更高
dotTrace 5.5 或更高
dotMemory 4.0 或更高
dotCover 1.0 或更高
RubyMine 1.0 或更高
PyCharm 1.0 或更高
WebStorm 1.0 或更高
PhpStorm 1.0 或更高
AppCode 1.0 或更高
CLion 1.0 或更高

### 使用方法
打开IntelliJ系列工具，菜单栏找到Help-Register,切换到License server ,License server address填写`http://intellij.mandroid.cn/` ,再Active即可，成功后界面如下：

该服务器仅供测试用，请购买正版软件
